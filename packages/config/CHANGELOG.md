# Changelog

All notable changes to the @nvlp/config package will be documented in this file.

## [1.0.0] - 2024-07-13

### Added
- Initial release of shared configuration package
- ESLint configuration with TypeScript and import ordering rules
- Prettier configuration with consistent formatting standards
- TypeScript base configuration with strict settings
- Jest configuration for TypeScript testing
- Dual format support (ESM and CommonJS) for all configurations
- TypeScript declaration files for proper type checking
- Comprehensive documentation and usage examples
- Support for React Native, Node.js/API, and library package configurations

### Features
- **ESLint**: TypeScript support, import ordering, extensible rules
- **Prettier**: Consistent code formatting across monorepo
- **TypeScript**: Strict mode, modern ES features, declaration output
- **Jest**: TypeScript testing, coverage reporting, flexible setup
- **Exports**: Clean package.json exports for easy importing
- **Documentation**: Complete usage guide with examples for different project types
