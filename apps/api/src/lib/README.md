# API Libraries

This directory contains shared utilities and libraries for the NVLP API.

## Structure

- `types/` - TypeScript type definitions
- `utils/` - Utility functions
- `lib/` - Shared library code

## Note

The actual Supabase Edge Functions are located in `/supabase/functions/` at the repository root.
This follows Supabase CLI conventions and ensures proper deployment.

The API package provides:
- Build and deployment scripts
- Documentation
- Type generation
- Shared utilities (if needed)
